**Step 2 — Carica la cartella dell'edizione**

Ora sei nella repo con il README. Da qui:

1. Clicca **"Add file"** → **"Upload files"**
2. Dal tuo computer, apri la cartella `didacta` che hai scaricato da qui
3. **Trascina l'intera cartella `didacta-firenze-2026`** nella zona di upload di GitHub (accetta cartelle intere, manterrà la struttura)
4. Attendi che tutti i file vengano caricati (index.html, data.json, aggiorna_dati.py, README.md + tutti i PNG nella sottocartella loghi)
5. Clicca **"Commit changes"**

---

**Step 3 — Attiva GitHub Pages**

1. Vai in **Settings** (⚙️ in alto nella repo)
2. Nel menu a sinistra clicca **"Pages"**
3. Sotto **"Source"** seleziona **"Deploy from a branch"**
4. Seleziona branch **`main`** e cartella **`/ (root)`**
5. Clicca **"Save"**
